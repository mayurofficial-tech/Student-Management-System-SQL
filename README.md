# 🎓 Final SQL Project - Student Management System

## 📌 Description
This project is a SQL-based Student Management System that demonstrates advanced database operations and data analysis techniques.

It manages students, departments, courses, enrollments, and instructors while performing various real-world queries.

## 🎯 Objective
The objective of this project is to design a relational database system and apply advanced SQL concepts such as joins, subqueries, window functions, and case expressions.

## 🧱 Database Structure
The database consists of the following tables:
- Students
- Departments
- Courses
- Enrollments
- Instructors

Each table is connected through foreign key relationships to maintain data integrity.

## 🚀 Features
- Database creation and management
- Insert, update, delete operations (CRUD)
- Multiple joins:
  - INNER JOIN
  - LEFT JOIN
- Subqueries for filtering data
- Aggregate functions and grouping
- Window functions (running total)
- Case statements for categorization
- Date and string functions

## 🛠️ Technologies Used
- SQL (MySQL)
- Relational Database Design
